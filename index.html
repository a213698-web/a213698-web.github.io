4. Reflection & Learning Journey

Growth from Lab 1 to Project 2

My technical journey evolved dramatically from simply arranging static text fields in Lab 1 to constructing a fully functional dynamic application equipped with its own Room database and live external cloud servers via Retrofit in Project 2. This has greatly enhanced my critical thinking and architectural approach to coding mobile systems.

Challenges faced

The biggest hurdle was resolving database schema migrations and properly configuring Git identity configurations, alongside ensuring flawless data synchronization transitions between web APIs (Live mode) and local fallback structures (Offline mode) without app crashes.

By engineering this automated healthcare application, a direct contribution to SDG 3 was achieved, providing the community with a systematic, user-friendly, and completely free alternative for self-health and medication tracking.
